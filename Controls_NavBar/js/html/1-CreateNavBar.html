The NavBar is hidden by default and appears when users either swipe a finger from the top edge of the screen, right-click or Windows key + Z. It overlays the content of the application and can be dismissed by the user with an edge swipe or by interacting with the application.

Implementing a NavBar involves three controls: NavBar, NavBarContainer and NavBarCommand. This scenario shows how to add NavBarCommands declaratively to a NavBarContainer.
