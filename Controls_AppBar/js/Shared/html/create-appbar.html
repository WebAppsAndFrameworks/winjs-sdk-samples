For PC, the WinJS.UI.AppBar is hidden by default and appears when users swipe a finger from the bottom edge of the screen. It covers the content of the app and can be dismissed by the user with an edge swipe or by interacting with the app.

For Phone, the AppBar is visible by default and has two states: closed and opened. A closed AppBar can be either "minimal" or "compact". This scenario shows an AppBar with "minimal" closed state. An AppBar goes to the opened state when the user taps the ellipsis on the AppBar, which displays the labels for the icon buttons and menu items (if enabled).
