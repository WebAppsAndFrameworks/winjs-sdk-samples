This scenario demonstrates asynchronous and synchronous renderers for the Listview itemTemplate and their effect on performance. The sample uses either a virtualized datasource, or a Binding.List datasource, both retrieving images from Bing search.

The Bing API is part of the Windows Azure Marketplace and requires an account key to identify the application. To use this sample you will need to get a key from the Bing Search API. Then paste the key into the text box below.
