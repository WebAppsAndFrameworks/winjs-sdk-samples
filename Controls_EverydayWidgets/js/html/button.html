There are three types of buttons: submit button, reset button, and standard button. The Submit button is activated when a user presses the Enter key while the focus is on another control in the same form that isn't a button, like a text field.

Clicking the reset button returns a field to its default state. In this example, it returns the field to its original text content.

The standard button can be set to call a JavaScript function. In this example, the button calls a simple arithmatic function.
