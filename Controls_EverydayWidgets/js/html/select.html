Select control can display in either drop-down mode, or multi-line mode. When a list is too long, it becomes scrollable. When a user comes to the end of the list, rather than having to scroll back up, the list will automatically wrap around and show the list from the beginning. This can reduce the amount of panning gestures needed to get to an item.
