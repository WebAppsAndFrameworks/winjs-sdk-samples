The following sample demonstrates how to use the Repeater Control with declarative WinJS.Binding.Templates to build a simple bar graph using a series of progress elements. In this example items can be dynamically added and removed from the graph and the Repeater control will automatically update the view and execute animations. Also the Repeater control can be used with CSS layouts like CSS Flexbox and CSS Grid to have full control over the graph's layout of items.
