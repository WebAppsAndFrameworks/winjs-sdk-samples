In this scenario we demonstrate how to enable dropping an HTML element into a specific location within the listview. This is done using listview specific drag and drop APIs.

In this case the drag source is the WinJS ItemContainer on the left, but the scenario will work with any HTML draggable element. This element can be dropped in the listview on the right, showing how to insert an element from another source / list into the listview as an item.
