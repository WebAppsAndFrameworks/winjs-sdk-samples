This scenario shows how to use a resource file to localize labels and tooltips for WinJS.UI.AppBarCommand controls. UI strings are placed into a separate string table (resources.resjson) so that they can be maintained and translated separately.

To see new resources on PC, do the following to change the language:

  • Go to the Desktop and choose Control Panel, Language. Add a language that is supported by this sample and then move the new language to the top of the list.
  • Languages supported: English (United States), French (France), German (Germany).

To see new resources on Phone, do the following to change the language:

  • Go to Settings, language. Add a language that is supported by this sample.
  • Languages supported: English (United States), French (France), German (Germany).
