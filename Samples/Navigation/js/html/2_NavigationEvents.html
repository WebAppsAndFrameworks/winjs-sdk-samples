The WinJS.Navigation system provides three events: beforenavigate, navigating, and navigated. These events let you control when navigation occurs, prepare your code to be navigated away from, and respond to a completed navigation.

This page lets you set various options and navigate between the scenarios of this sample. In addition, the scenario selector list also uses the navigation framework; please see the event handler for navigated in default.js for details of how the contents of the screen are changed upon navigation.
