This scenario demonstrates how to create a data adapter to interface with a web service. A data adapter is the component to interface between the list view and the supply of data, enabling virtualization. This specific example uses XmlHttpRequest to access the image search feature of Bing which is exposed as a web service.

The Bing API is part of the Windows Azure Marketplace and requires an account key to identify the application. To use this sample you will need to get a key from the Bing Search API. Then paste the key into the text box below.
