The default style sheet also contains CSS style rules that will mirror HTML elements bi-directionally. This happens automatically as the system will mark the root element with the application language.

If an application needs to set this explicity, it can be done by setting the dir attribute and lang attribute of the containing element.

Ensure that the application links in ui-light.css or ui-dark.css files for UI mirroring support (Note: This sample links to ui-light.css by default).

Please place the system in en-US or ar-SA to mirror HTML elements in the "Application set direction" column.
