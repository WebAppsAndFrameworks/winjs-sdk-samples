This sample demonstrates how to use incremental styling to override the accent colors of the various controls provided for applications.

  1. Select a light or dark theme
  2. Enter the RGB values for: Primary Color, Primary Text Color, Secondary Color, and Secondary Text Color
  3. Click the "Apply color" button to apply the defined colors to the HTML elements below
