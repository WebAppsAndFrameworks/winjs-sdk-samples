This scenario demonstrates how to use the ItemContainer to create a small set of draggable items that respond to press and tap. Notice the pressed animation on tap/click, and that the win-dragsource CSS class is applied while dragging (providing a default, semi-transparent style to the original item). All of this comes "for free" to developers using the ItemContainer!
